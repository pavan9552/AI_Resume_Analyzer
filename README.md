# AI_Resume_Analyzer
An intelligent recruitment tool built with React, React Router, and Puter.js that helps recruiters and HR teams streamline candidate screening.
💡 Key Features

📋 Job Listings – Create and manage job postings with detailed requirements.

📂 Resume Upload – Candidates or recruiters can upload resumes in PDF/DOC formats.

🤖 AI Matching – Automatically analyze resumes using AI to match skills, experience, and qualifications with job descriptions.

📊 Score & Feedback – Get AI-generated compatibility scores and improvement suggestions.

⚡ Instant Search & Filter – Quickly find the best-fit candidates.

🛠 Tech Stack

Frontend: React, React Router

Backend & Storage: Puter.js (Serverless, persistent data, and file storage)

AI Processing: Integrated AI APIs for resume parsing and job matching logic

📌 Use Case
Perfect for HR teams, startups, and recruiters looking to save hours of manual resume screening and improve hiring efficiency.
